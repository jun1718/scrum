# 12. 에러·유효성 메시지

FE/BE/QA 공통으로 사용할 메시지. 코드/화면 문구 확정 시 통일.

| 구분 | 메시지 | 비고 |
|------|--------|------|
| 다중 팀 미지원 | 현재 스프린톤에서는 다중 팀 기능을 지원하지 않습니다. 추후 개선될 예정입니다. | 팀 지정 시 이미 Member 존재 |
| 성과 기간 중복 | 이미 생성된 시작~종료일 범위와 겹칩니다. | 성과 보고 생성 |
| 한 일 필수 | 해당 업무로 한 일은 필수입니다. (공백/빈문자 불가) | 데일리 슈크럼 |
| 업무 링크 형식 | 업무 링크는 /project/tasks/ 숫자 형태여야 합니다. | taskLink 검증 |
| 로그인 실패 | 로그인에 실패했습니다. 토큰을 확인해 주세요. | POST /login 실패 |
| JWT 만료/무효 | 인증이 만료되었거나 유효하지 않습니다. 다시 로그인해 주세요. | 401 |

추가 메시지는 위 표에 행으로 추가.
