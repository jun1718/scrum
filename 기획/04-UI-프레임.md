# 4. UI 프레임 설명

**참고**: 행동별 상세(화면별 필드·저장 규칙)는 **08-사용자-행동별-기능정의.md** 참고.

---

## 좌측 메뉴 (메뉴 버튼 세로 정렬)

- **팀 생성하기** | **팀 시작/종료일자 설정하기** | **태그 설정하기** | **팀 지정하기**(팀 미지정 시에만 노출) | **내 업무보고 확인(메인)** | **팀 업무 공유**
- **내 업무보고 확인**이 최초 접근 시 기본 선택.

**팀장용 화면 (클릭 시)**  
- **팀 생성하기**: 팀명 입력 폼 → 저장.  
- **팀 시작/종료일자 설정하기**: 주간 보고 **시작 요일·종료 요일** 설정 (예: 수~화).  
- **태그 설정하기**: 주간 보고용 태그 먼저 지정 → 그다음 월간(ESM) 그룹핑용 태그 지정.

**팀원용**  
- **팀 지정하기**: 팝업 — 팀 이름 입력(자동 검색) 또는 목록에서 선택 → 자신을 해당 팀에 배정. **팀이 이미 지정된 경우 좌측 메뉴에서는 숨김**(상단 헤더에 현재 팀명만 표시).

---

## 팀 미지정 시 메인 화면

- 좌측 메뉴는 동일. 중앙(메인 영역)에 **"팀을 지정해주세요"** 문구 노출.
- **오늘 한 일** 업무 목록 영역은 **미표시**. 팀 지정 완료 후 새로고침 시 "내 업무 보고" 플로우로 전환.

---

## 내 업무 보고 메뉴

### 상단 5개 탭

`데일리 슈크럼 작성 | 데일리 보고 | 주간 보고 | 월간 보고(GRM) | 성과 보고`  
(한글 주 표기, 영어·출처는 08 상단 탭 테이블 참고.)

- **데일리 슈크럼 작성**이 기본 선택 탭. (없으면 생성, 있으면 당일 저장분 노출.)
- 각 탭은 **자신의 보고 데이터를 최신 일자순**으로 노출.
- 주간/월간 보고는 **배치 시스템이 자동 생성**하며, 이후 본인이 **수정/삭제/재생성** 가능.
- 보고 생성 계층: 데일리 스크럼 → (배치) 데일리 보고 → 주간 보고 → 월간 보고 → (사용자 버튼) 성과 보고.
- **성과 보고 탭**: 월간 보고 데이터 이용, 사용자 버튼 클릭 시 ESM별 AI 요약. 날짜 지정 후 "성과 생성" 버튼 (기본 1/1~12/31). 기간 겹치면 오류.

### 탭별 화면 구성

| 탭 | 화면 구성 |
|----|-----------|
| **데일리 슈크럼 작성** | 당일 입력 폼. 업무 목록 + 행 단위 입력 + **+** 버튼(직접 추가) / **X** 버튼(행 제거) / **동료 협업 기록** 버튼 / **저장** 버튼. |
| **데일리 보고** | 자신이 작성한 데일리 슈크럼 **최신순 목록**. 수정/삭제/생성 가능. |
| **주간 보고** | 주간 보고 목록. 수정/삭제/생성 가능. |
| **월간 보고(GRM)** | 월간 보고 목록. 수정/삭제/재생성 가능. |
| **성과 보고** | 성과 보고 목록. 날짜 지정 후 **성과 생성** 버튼으로 생성. |

### 데일리 슈크럼 작성 탭 — 세부 UI (대시보드 그리기용)

- **업무 목록 소스(당일 기준)**: "내가 멘션된 업무", "내가 쓴 댓글", "내가 등록자/담당자/참조자이면서 수정된 업무".
- **행 단위**: 업무 제목, 해당 업무로 한 일(필수), 투입시간, 어필할 성과, 업무 링크, **주간 보고 태그**(필수) 선택. **주간 보고 태그 미선택 시 저장 불가.** 월간 보고 태그는 주간 태그의 부모 태그(parentTagId)로 시스템이 자동 매핑.
- **+ 버튼**: 링크·내용 직접 추가(시스템 목록 외 행 추가).
- **X 버튼**: 해당 행 제거. 저장 전이면 UI만 제거, DB 저장된 경우 삭제 API 호출 후 UI 제거.
- **동료 협업 기록**: 버튼 → 팀 소속 동료 이름 목록 → 이름 선택 → 협업 내용 작성 → 저장.
- **저장 버튼**: 저장 후에도 행 수정·행 삭제 가능.

## 팀 업무 공유 메뉴

- **상단 3개 탭**: `데일리 보고 | 주간 보고 | 월간 보고(GRM)`
- 해당 팀 소속 팀원들의 **일자별 보고 목록** 노출 (투입률 %, 총 투입시간). **투입률(%)** = 특정 업무 workHours / 당일 전체 workHours 합계 × 100.
- 최초 일자는 **당일**. **민감 정보 비노출** (어필할 내용, 동료 협업 기록 등).
